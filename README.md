# 🚀 Student TaskPilot

> Your personal academic command center for tasks, goals, focus, and productivity.

Student TaskPilot is a modern student productivity dashboard built with pure HTML, CSS, and JavaScript. It helps students organize their academic life by combining task management, goal tracking, focus sessions, note-taking, and productivity analytics into a single elegant interface.

---
Students often switch between multiple apps for notes, tasks, timers, and study planning.

Student TaskPilot brings everything together in one clean and responsive workspace.

### Built For

* College Students
* University Learners
* Competitive Exam Aspirants
* Self-Learners
* Productivity Enthusiasts

---

## 🎯 Core Features

### 📋 Smart Task Manager

* Create new tasks
* Mark tasks as completed
* Delete tasks
* Priority levels (High, Medium, Low)
* Search tasks instantly
* Local storage persistence

### 🎯 Goal Tracker

* Daily study goals
* Weekly study goals
* Visual progress indicators
* Goal completion tracking

### ⏱ Focus Mode

* 25-minute Pomodoro sessions
* Start, Pause, and Reset controls
* Session tracking
* Productivity boosting workflow

### 📝 Notes Workspace

* Quick note-taking
* Auto-save functionality
* Export notes as text files

### 📊 Productivity Dashboard

* Dynamic greetings
* Daily motivational quotes
* Productivity score calculation
* Study streak tracking
* Progress visualization

### 🌙 Dark Mode

* One-click theme switching
* Theme preference saved automatically

### 🏆 Achievement System

Unlock milestones as you progress:

* 🚀 Beginner
* 📚 Study Streak
* ✅ Task Master
* ⚡ Focus Champion
* 🏆 Productivity Pro

## 🚀 Getting Started

```bash

```

